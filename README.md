# EcoGuard
EcoGuard: A modern hub for climate awareness and education. Learn about renewable energy, biodiversity, and sustainable habits through a sleek, glassmorphism-inspired interface. Built with HTML, CSS, and JS, this platform provides actionable tips and data to help you build a more sustainable future.
